BookSpot

Finding the Best Location for a New Bookstore

BookSpot is a GIS-based project that aims to identify suitable locations for establishing a new bookstore. This project applies spatial analysis by considering accessibility, nearby educational facilities, commercial areas, and the distribution of existing bookstores.

About Project

This website was created as part of the WebGIS Development Bootcamp assignment using HTML and CSS.
